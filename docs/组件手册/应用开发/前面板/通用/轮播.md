# 功能说明

轮播组件用于滚动展示若干图片。

## 配置项

| 配置项               | 说明                                                   |
| -------------------- | ------------------------------------------------------ |
| 定时器时间间隔（秒） | 设置图片切换的频率，以秒为单位                         |
| 面板指示点背景色     | 轮播图底部的页码，缺省为透明                           |
| 面板当前指示点背景色 | 当前处于激活状态的轮播图对应的页码点的颜色，缺省为黑色 |
| 面板指示点边框颜色   | 轮播图底部页面的边框颜色                               |
