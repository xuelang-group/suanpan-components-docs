## 组件原理
将多个文件夹合并成一个文件夹。
## 输入桩
支持多个files文件输入。
### 输入端子1

- **端口名称：** 文件夹1
- **输出类型：** files文件
- **功能描述：** 需要合并的文件夹

## 输出桩
支持folder文件输出。
### 输出端子1

- **端口名称：** 输出1
- **输出类型：** folder文件
- **功能描述：** 输出合并的文件夹

## 使用方法
- 加组件拖入到项目中
- 与前一个组件输出的端口连接
- 点击运行该节点
